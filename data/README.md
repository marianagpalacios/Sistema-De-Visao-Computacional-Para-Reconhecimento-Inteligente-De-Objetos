# Dataset

O dataset utilizado neste projeto contém 360 imagens, tiradas durante os teste, distribuídas entre classes relacionadas ao uso de EPIs:

- capacete;
- óculos de proteção;
- máscara;
- combinações entre os EPIs;
- ausência de equipamentos.

As imagens foram redimensionadas para 224x224 pixels e normalizadas no intervalo [0, 1].

Por questões de privacidade e tamanho dos arquivos, o dataset completo não está incluído neste repositório.
